# Code of Conduct

## TL;DR

Be kind. Be honest. Don't drama. Help others. Disagree respectfully.

## Expected behavior

- Use welcoming and inclusive language
- Respect differing viewpoints and experiences
- Accept constructive feedback gracefully
- Focus on what's best for the project and its users
- Show empathy toward other contributors

## Unacceptable behavior

- Harassment in any form
- Personal attacks or insults
- Trolling or deliberately inflammatory comments
- Public or private harassment
- Publishing others' private information without permission
- Other conduct that would be inappropriate in a professional setting

## Enforcement

Issues or behavior violating this code can be reported to the project maintainer at the email in [LICENSE](LICENSE) or by opening a private issue. All reports will be reviewed and acted on confidentially. Maintainers may remove comments, ban accounts, or close issues that violate this code.

## Scope

This code applies in all project spaces — GitHub issues, PRs, discussions, and any project communication channels — and when an individual represents the project in public.
